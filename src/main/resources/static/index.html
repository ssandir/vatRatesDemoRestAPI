Welcome to my JSON query REST API

The API originally works on the format of http://jsonvat.com/, but can be extended to accommodate other formats.

The table below gives parameters and their descriptions. Possible values are given in tables below.

Query address is http://localhost:8080/query.

Parameter Description
s Source of the jason to query, default value is 0
f Specify function for filtering
p Parameters to the filter function

Source location table

Value of s Meaning
0 http://jsonvat.com/

Filter function table

Value of f Meaning
lv Given parameter p, returns p countries with lowest VAT rates, default is p=3
hv Given parameter p, returns p countries with highest VAT rates, default is p=3

Output is in the format JSON {"status":status, "result":resultJSON}, where status is either "success" or "error".


Example outputs

Query Result
http://localhost:8080/query?f=hv {"status":"success","result":["Sweden","Denmark","Hungary"]}
http://localhost:8080/query?f=lv {"status":"success","result":["Cyprus","Luxembourg","Malta"]}
http://localhost:8080/query?f=lv&p=1 {"status":"success","result":["Luxembourg"]}
http://localhost:8080/query?f=lv&p=6 {"status":"success","result":["Cyprus","Germany","Luxembourg","Romania","Malta","United Kingdom"]}
